# Primeiro app de um jeito fácil
Como sabemos, vamos ciar apps, e algumas pessoas não tem um computador robusto para programação nativa.
Podemos criar emulando em nossos celulares, e o React Native (Expo) nos dá essa opção.

### Para começar a programar apps conosco, você deve ter:

1. [O que é o Expo?](https://youtu.be/uxZe6TGsVJk?si=UjW3oifR1NvGw2Wv)
2. [Visual Studio Code](https://code.visualstudio.com/)
3. [Expo Go no celular](https://expo.io/client)
4. [Node](https://nodejs.org/)
5. [Expo criar primeiro projeto](https://docs.expo.io/get-started/create-a-new-app/)

### COMO RODAR OS APPS?
Vá para a pasta que tem os aquivos
`cd caminho_novo/

Rode com o node ou expo e siga as instruções em seu terminal:

`npm start`

`expo start`
