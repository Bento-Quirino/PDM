## Instruções Gerais para os Grupos

- **Cada grupo deve ter até 5 integrantes**.
- **Gerar um APK** para cada projeto, incluindo um **ícone** e uma **tela de splash** personalizada.
- Criar um **README** com as seguintes informações:
  - O que é o projeto
  - Bibliotecas usadas
  - Nome de todos os integrantes do grupo
  - Vídeo curto do app rodando
