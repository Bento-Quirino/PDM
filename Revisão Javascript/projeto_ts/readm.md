## Criando esse projeto

- Primeiro, crie uma pasta para o projeto (ex: `projeto_ts`)
- Depois, entre na pasta e execute o comando `npm init -y` para criar o arquivo `package.json` com configurações padrão
- Em seguida, crie um arquivo index.js
- Agora, instale o TypeScript com o comando `npm install typescript`
- Crie um arquivo de configuração do TypeScript com o comando `npx tsc --init`
- Agora com o arquivo `tsconfig.json` criado, você pode configurar o TypeScript como quiser
- Para compilar o arquivo `index.js` para `index.ts`, execute o comando `npx tsc index.ts`
